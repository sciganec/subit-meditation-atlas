# SUBIT · Meditation Atlas

> **PRIME · SYNC · SCAN · CORE · {0,1}⁶ · 64 архетипи**  
> Буддизм · Індуїзм · Даосизм

Атлас медитативних практик на основі фреймворку **SUBIT** — 6-бітної системи координат що відображає практики через три осі: суб'єктну позицію (WHO), просторовий вектор (WHERE) і часову якість (WHEN).

---

## Чотири режими

Ядро SUBIT — чотири режими, кожен з яких є **тотожністю** між WHO, WHERE і WHEN:

```
|PRIME| : |ME|,   |EAST|,  |SPRING|, |10|
|SYNC|  : |WE|,   |SOUTH|, |SUMMER|, |11|
|SCAN|  : |YOU|,  |WEST|,  |AUTUMN|, |01|
|CORE|  : |THEY|, |NORTH|, |WINTER|, |00|
```

| Режим | WHO  | WHERE | WHEN   | Bits | Якість                       |
|-------|------|-------|--------|------|------------------------------|
| **PRIME** | ME   | EAST  | SPRING | `10` | Індивідуальне · Висхідна · Весна |
| **SYNC**  | WE   | SOUTH | SUMMER | `11` | Колективне · Трансформуюча · Літо |
| **SCAN**  | YOU  | WEST  | AUTUMN | `01` | Реляційне · Низхідна · Осінь |
| **CORE**  | THEY | NORTH | WINTER | `00` | Архетипне · Стала · Зима     |

## Три осі

```
SUBIT = WHO × WHERE × WHEN

WHO   ∈ { ME, WE, YOU, THEY }              — b₁b₂
WHERE ∈ { EAST, SOUTH, WEST, NORTH }       — b₃b₄
WHEN  ∈ { SPRING, SUMMER, AUTUMN, WINTER } — b₅b₆

|SUBIT| = |{0,1}⁶| = 64 архетипи
```

Кожен архетип — 6-бітний рядок що розкладається у два тригрими (▲ верхній + ▽ нижній), утворюючи колірну систему з 8 × 8 матриці.

## Структура атласу

### Атлас — 64 картки
- Фільтрація за WHO / WHERE / WHEN / традицією
- Колірне кодування: режим (PRIME/SYNC/SCAN/CORE) + традиція + тригрими
- Sidebar: швидкі фільтри по всіх трьох осях
- Права панель: повна практика з історичним контекстом

### Кожна практика містить
- **Назва** і **тег-лайн**
- **Традиція і лінія передачі** — джерело, вчителі, дати
- **Історичний контекст** — першоджерело, нейронаукові дані
- **Інструкція** — покрокова техніка від другої особи
- **Мантра** / фокусна фраза
- **4 варіанти** і лінії передачі
- **Тривалість і поза**

### Теорія — 5 секцій
1. **Таксономія** — 7 родів медитативних практик
2. **Осі SUBIT** — повна карта режимів PRIME/SYNC/SCAN/CORE
3. **Буддизм** — 6 практик з контекстом
4. **Індуїзм** — 6 практик з контекстом
5. **Даосизм** — 5 практик з контекстом

## Запуск

Атлас — standalone HTML-файл. Не потребує сервера, бекенду або інтернету.

```bash
git clone https://github.com/sciganec/subit-meditation-atlas.git
cd subit-meditation-atlas

# Відкрити
start src/atlas.html        # Windows
open src/atlas.html         # macOS
xdg-open src/atlas.html     # Linux
```

### GitHub Pages

Після налаштування Pages (Settings → Pages → Deploy from branch → main) атлас доступний за адресою:

```
https://sciganec.github.io/subit-meditation-atlas/
```
